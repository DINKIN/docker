# docker

> Docker image for `thumbsup`

Includes:

- [thumbsup](https://www.npmjs.com/package/thumbsup) npm package
- ffmpeg
- graphicsmagick

## Building the image

```bash
./build
# Successfully built thumbsup/thumbsup
docker push thumbsup/thumbsup
```
